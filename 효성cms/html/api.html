연동서비스

효성에프엠에스는 다양한 API를 통한
수납환경을 제공합니다.

효성에프엠에스는 고객사의 프로그램에
결제서비스를 연동하기 위한
다양한 IT 기술 및 인프라를 제공합니다

Web API
SDSI
DB Agent

Web Application Programming Interface

고객사(Client)의 프로그램에서 손쉽게
사용할 수 있도록
표준 HTTP 통신
프로토콜을 활용한 API 주소를 제공하는
연동 방식입니다

모든 데이터는 SSL 규약으로 암호화되며
JSON 형태로 전송됩니다

개요

표준 HTTP 통신 프로토콜을 활용한 API 서비스

인증

표준 HTTP 통신 Authorization 헤더 사용

보안

모든 통신은 SSL 규약으로 안전하게 데이터 전송

형태

가독성이 높은 개방형 표준 형식인 JSON 방식 사용 속성(Key, Value)쌍의 형태를 지원해 프로그램 언어의 제약이 없음

Standard Data Socket Interface

효성에프엠에스만의 표준화된 Socket
통신 방식으로
고객사(Client)와 효성에
프엠에스(Server) 간 암호화된 전문을
송수신하는 연동 방식입니다

모든 데이터는 암호화 통신으로 규격화된
전문을 주고 받습니다

개요

TCP/IP 프로토콜을 활용한 서비스

인증

효성에프엠에스에서 발급한 고객사 고유의 인증정보 사용

보안

암호화 라이브러리를 이용한 전문 데이터 송수신

형태

규격화된 전문 사용

Web Application Programming Interface

별도의 개발없이 고객사(Client) 서버에
DB Agent 설치 및 환경 설정 만으로
DB 정보를 조회해 효성에프엠에스
(Server)로 자동으로 처리해주는
연동 방식입니다

모든 데이터는 암호화 통신으로 규격화된
전문을 주고 받습니다

개요

TCP/IP 프로토콜을 활용한 서비스

인증

효성에프엠에스에서 발급한 고객사 고유의 인증정보 사용

보안

암호화 라이브러리를 이용한 전문 데이터 송수신

형태

규격화된 전문 사용 *설치 패키지 종류 : Windows, Linux, Unix 사용 가능
